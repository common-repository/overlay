== Installation ==

To do a new installation of the plugin, please follow these steps

1. Download the zipped plugin file to your local machine.
2. Unzip the file.
3. Upload the `overlay` folder to the `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Go to the Overlay Plugin and save the settings!

To use, read http://www.fishme.de/overlay


If you have already installed the plugin

1. De-activate the plugin.
2. Download the latest files.
3. Follow the new installation steps.
